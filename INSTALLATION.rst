============
Installation
============

At the command line::

    $ pip install toonapilib

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv toonapilib
    $ pip install toonapilib

Or, if you are using pipenv::

    $ pipenv install toonapilib

Or, if you are using pipx::

    $ pipx install toonapilib
